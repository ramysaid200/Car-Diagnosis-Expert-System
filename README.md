# Car Diagnosis Expert System

## Overview
This project is a rule-based Expert System designed to diagnose common car malfunctions. By asking the user a series of targeted yes/no questions about the symptoms their vehicle is experiencing, the system navigates a logic tree to deduce the root cause of the problem (e.g., battery failure, alternator issues, starter motor faults, or fuel system errors) and provides actionable solutions.

## Key Features
* **Rule-Based Architecture:** Utilizes a robust knowledge base of facts and rules to simulate the decision-making ability of a human mechanic.
* **Interactive Diagnostics:** Engages the user in a dynamic Q&A session, pruning the search space based on symptom confirmation or denial.
* **Knowledge Representation:** Symptoms, components, and vehicle states are mapped logically to ensure accurate and reliable diagnostic conclusions.
* **Extensibility:** The knowledge base is modular, allowing for the easy addition of new car models, symptoms, and diagnostic rules without altering the core inference engine.
